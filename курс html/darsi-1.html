Урок 1: Введение в HTML

HTML (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц. Он описывает, как должен отображаться контент в браузере.

Основная структура HTML-документа

<!DOCTYPE html>
  <html lang="ru">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Пример страницы</title>
    </head>
    <body>
      <h1>Привет, мир!</h1>
    </body>
  </html>

Ключевые теги HTML

  • <html> — корневой элемент HTML-документа.
  • <head> — содержит метаинформацию о документе.
  • <title> — заголовок страницы в браузере.
  • <body> — отображаемая часть страницы (всё, что видит пользователь).

Что нужно для начала?

Чтобы начать писать HTML-код, тебе нужен:

  1. Текстовый редактор (например, VS Code).
  2. Браузер (Chrome, Firefox и т.д.) для просмотра результатов.
  3. Желание учиться!

Теперь ты знаешь, как выглядит базовая HTML-страница. Переходи ко второму уроку, чтобы изучить основные теги!
